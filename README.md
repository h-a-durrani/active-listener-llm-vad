# Active-listener-LLM-VAD
Real-time voice assistant with barge-in support for service robotics
Speak naturally — interrupt the bot mid-sentence just like a human conversation.

## How it works
- Microphone is always listening using WebRTC VAD
- Speech is transcribed offline using OpenAI Whisper
- Response is generated by TinyLlama running locally on-device
- Text-to-speech via macOS `say` command
- Barge-in: if you speak above a dB threshold while the bot is talking, it stops immediately

## Setup
pip install -r requirements.txt
python Active_Listener_LLM_VAD.py

## Hardware
Tested on MacBook Pro M1 (Apple Silicon). Runs fully offline — no internet required after model download.
